# CAD Viewer - egui 前端

CAD 几何智能处理系统的图形用户界面前端。

## 功能特性

### 已实现功能（P0 优先级）

- ✅ **Canvas 渲染** - 线段绘制、缩放、平移
- ✅ **鼠标点选边** - 射线检测 + 容差选择
- ✅ **实时高亮** - 选中边黄色高亮显示
- ✅ **自动追踪** - 从选中边开始追踪路径
- ✅ **缺口检测** - 检测并可视化显示缺口（红色虚线）
- ✅ **圈选工具** - 右键绘制多边形选择区域
- ✅ **文件操作** - 打开 DXF/PDF 文件、导出 JSON 场景
- ✅ **视图控制** - 缩放、平移、重置视图
- ✅ **日志控制台** - 实时显示操作日志

### 计划功能（P2 阶段）

- ⏳ WebSocket 实时通信
- ✅ 后端 API 集成（2026 年 2 月完成）
- ⏳ 语义标注编辑
- ⏳ 圈选区域提取环
- ⏳ 缺口补全工具

## 构建和运行

### 构建

```bash
cargo build -p cad-viewer
```

### 运行

```bash
cargo run -p cad-viewer
```

### Release 版本

```bash
cargo build -p cad-viewer --release
```

## 交互操作

| 操作 | 鼠标/键盘 | 说明 |
|------|----------|------|
| 选择边 | 左键点击 | 选中线段（黄色高亮） |
| 平移视图 | 中键拖动 | 移动画布 |
| 缩放视图 | 滚轮滚动 | 放大/缩小 |
| 开始圈选 | 右键点击 | 开始绘制多边形 |
| 结束圈选 | 右键点击 | 完成圈选 |
| 重置视图 | 按钮点击 | 恢复默认缩放和平移 |

## 技术栈

- **GUI 框架**: egui 0.29 + eframe 0.29
- **渲染**: egui 自定义 Canvas 组件
- **文件对话框**: rfd 0.15
- **HTTP 客户端**: reqwest 0.12（预留）
- **异步运行时**: tokio 1.0
