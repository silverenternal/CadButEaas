## 一、支持的文件类型

### ✅ 矢量 PDF（支持）

**来源**:
- AutoCAD 导出（`PLOT` → 选择 PDF 打印机）
- LibreCAD 导出
- 其他 CAD 软件导出的矢量 PDF

**特征**:
- PDF 内部包含直线、圆弧、多段线等矢量图元
- 文件体积小（通常 < 1MB）
- 放大后边缘清晰，无锯齿

**解析方式**:
```rust
use parser::DxfParser;

let mut parser = DxfParser::new();
let entities = parser.parse_pdf_file("floor_plan.pdf")?;
```

---

### ✅ 扫描版 PDF 矢量化（已完成）

**来源**:
- 扫描仪扫描的纸质图纸
- 手机拍照转 PDF
- 截图保存为 PDF
- 光栅图像嵌入 PDF

**特征**:
- PDF 内部是位图图像（如 PNG、JPEG）
- 文件体积大（通常 > 5MB）
- 放大后有锯齿、模糊

**处理方案**:
系统已内置自动矢量化功能，无需第三方工具转换！

**技术实现**:
1. **图像预处理**: 去噪、对比度增强、自适应阈值
2. **边缘检测**: Sobel 算子、骨架化
3. **轮廓提取**: 迭代 DFS（避免栈溢出）
4. **矢量化**: Douglas-Peucker 简化（迭代实现）
5. **质量评估**: 自动评分 + 警告

**性能参考**:
- 500x500 像素：~50ms
- 1000x1000 像素：~200ms
- 2000x2000 像素：~800ms

**限制**:
- 最大支持 250 万像素（约 1581x1581）
- 推荐图像尺寸 < 2000x2000 像素
- 超大图像建议启用 OpenCV 加速

## 二、技术实现说明

### 扫描版 PDF 矢量化实现

扫描版 PDF 矢量化已完整实现，采用以下流程：

```
扫描 PDF → 图像预处理 → 边缘检测 → 线结构提取 → 矢量化 → 拓扑修复 → 质量评估
           ↓              ↓           ↓            ↓          ↓           ↓
        去噪/二值化    Sobel/骨架化  迭代 DFS    Douglas-Peucker  端点吸附  自动评分
```

**技术亮点**:
1. **迭代算法**: 所有递归算法已改为迭代实现，避免栈溢出
2. **堆分配优化**: 大图像使用堆分配，避免栈内存不足
3. **尺寸限制**: 自动检测图像尺寸，超过 250 万像素返回友好错误
4. **DPI 自适应**: 根据 DPI 自动调整处理参数
5. **质量评估**: 自动评分，低于阈值时警告

## 三、常见问题解答

### Q1: 如何判断我的 PDF 是矢量还是图片？

**方法 1**: 用 Adobe Reader 打开，放大到 500%
- 矢量：边缘清晰，无锯齿
- 图片：出现像素点，模糊

**方法 2**: 查看文件大小
- 矢量：通常 < 1MB
- 图片：通常 > 5MB

**方法 3**: 用文本编辑器打开
- 矢量：包含 `LINE`、`PATH` 等关键词
- 图片：包含大量乱码（二进制图像数据）

---
