## 一、系统概述

### 1.1 系统定位

CAD 几何智能处理系统是基于「一切皆服务」(EaaS) 设计哲学的工业级 CAD 几何智能处理系统，融合**计算机视觉、计算几何、图形拓扑建模、人机协同交互与工程仿真接口**。

系统核心目标：将客户提供的 CAD 图纸（DWG/DXF）或 PDF 平面图（矢量/图片）转换为可用于声学仿真的"可计算几何"。

### 1.2 核心价值

| 价值点 | 说明 |
|--------|------|
| **自动化** | 从图纸到仿真几何的全自动处理，减少人工干预 |
| **高精度** | 矢量解析精度达 0.1mm，拓扑构建 O(n log n) 复杂度 |
| **可追溯** | 220+ 测试全覆盖，错误恢复建议系统 |
| **可扩展** | 服务化架构，支持 HTTP/gRPC 微服务部署 |

## 二、核心功能模块

### 2.1 图纸解析服务 (Parser Service)

**功能描述**: 解析 DXF/PDF 文件，提取几何实体和语义信息

**支持的输入格式**:

| 格式 | 类型 | 支持实体 | 说明 |
|------|------|----------|------|
| **DXF** | 矢量 | LINE, LWPOLYLINE, ARC, CIRCLE, SPLINE, ELLIPSE, BLOCK/INSERT | AutoCAD R14 (AC1015) 及以上 |
| **PDF** | 矢量 | LINE, PATH, RECT, CURVE | 直接提取矢量图元 |
| **PDF** | 光栅 | 图像 | 自动矢量化（边缘检测 + 骨架化 + 轮廓提取） |

**核心能力**:

| 能力 | 技术指标 | 说明 |
|------|----------|------|
| 智能图层识别 | AIA 标准 + 中文变体 | 自动识别墙体/门窗/标注图层 |
| NURBS 离散化 | 弦高误差 < 0.1mm | 曲线精确转换为折线 |
| 嵌套块展开 | 递归深度无限制 | 支持块中块嵌套结构 |
| 单位解析 | 15 种单位自动识别 | mm/cm/m/inch/foot 等 |
| 颜色/线宽过滤 | 24 级线宽 | 按线宽过滤候选墙线 |

**性能指标**:
- 1000 实体 DXF: <100ms
- 541,216 PDF 实体：1.5s

### 2.2 图像矢量化服务 (Vectorize Service)

**功能描述**: 将光栅图像（扫描 PDF/截图）转换为矢量几何

**处理流程**:
```
图像预处理 → 边缘检测 → 骨架化 → 轮廓提取 → 矢量化 → 质量评估
```

**核心算法**:

| 算法 | 说明 | 性能 |
|------|------|------|
| 自适应阈值二值化 | Otsu 算法 | 自动适应光照变化 |
| Sobel/Canny 边缘检测 | 可选 OpenCV 加速 | 4.5x 性能提升 |
| Zhang-Suen 骨架化 | 迭代实现 | 避免栈溢出 |
| DFS 轮廓提取 | 迭代实现 | 稳定提取闭合轮廓 |
| Douglas-Peucker 简化 | 迭代实现 | 减少点数 90%+ |
| Kåsa 圆弧拟合 | 最小二乘法 | 精确识别圆弧 |
| R*-tree 端点吸附 | 空间索引加速 | O(n log n) 复杂度 |

**适用场景**:
- ✅ 线条清晰的建筑平面图（推荐）
- ✅ 扫描质量良好的图纸
- ✅ 高对比度的工程图

**限制说明**:
- ⚠️ 推荐图像尺寸 < 2000x2000 像素
- ⚠️ 严重阴影/折痕/褪色的图纸建议转 DXF

**性能指标** (OpenCV 加速):
| 像素 | 纯 Rust | OpenCV | 提升 |
|------|--------|--------|------|
| 2000×3000 | ~1000ms | ~220ms | **4.5x** |

### 2.3 拓扑建模服务 (Topo Service)

**功能描述**: 构建几何拓扑关系，提取闭合边界和孔洞

**核心算法**:

| 算法 | 复杂度 | 说明 |
|------|--------|------|
| 端点吸附 | O(n log n) | R*-tree 空间索引 |
| 交点切分 | O(n log n) | Bentley-Ottmann 扫描线 |
| 平面图构建 | O(n) | 节点 - 边 - 邻接表 |
| 闭合环提取 | O(n log n) | DFS + 夹角最小启发式 |
| 孔洞判定 | O(n) | 负面积 + 射线法 |

**输出内容**:
- 外边界（闭合多边形）
- 内部洞/障碍物（多边形集合）
- 节点 - 边拓扑关系

**性能基准**:
| 线段数 | 时间 | 每线段 |
|--------|------|--------|
| 100 | 13.4ms | 134μs |
| 500 | 67.6ms | 135μs |
| 1000 | 131.9ms | 132μs |

**复杂度验证**: O(n log n) ✅

### 2.4 交互协同服务 (Interact Service)

**功能描述**: 提供人机交互接口，支持边界确认和修复

**交互模式**:

#### 模式 A：选边追踪 (Edge Picking + Auto Trace)

用户点选一段墙线后，系统沿拓扑自动追踪形成闭合环：

1. 在每个节点选择"最可能延续"的下一条边
2. 优先小转角、同层/同属性
3. 遇到分叉时提示用户选择
4. 成功闭合则输出一个环

**API 调用序列**:
```
POST /interact/auto_trace { edge_id: 1 } → 返回追踪路径
WebSocket: select_edge { edge_id: 1 } ← 收到 edge_selected 事件
POST /interact/set_boundary_semantic { segment_id: 1, semantic: "hard_wall" }
```

#### 模式 B：圈选区域 (Lasso/Polygon Selection)

用户圈定区域后，系统在选区内提取闭合环：

1. 连通组件分析，提取闭合候选
2. 按面积与闭合度排序
3. 最大可用区域通常为主区域
4. 其它闭合环作为洞/障碍物候选

**API 调用序列**:
```
POST /interact/lasso { polygon: [[0,0],[20,0],[20,15],[0,15]] }
POST /interact/detect_gaps { tolerance: 2.0 }
POST /interact/snap_bridge { gap_id: 0 }
```

#### 缺口检测与分层补全

**缺口类型**:
| 类型 | 描述 | 处理策略 |
|------|------|----------|
| `collinear` | 共线缺口 | 自动桥接 |
| `orthogonal` | 正交缺口 | 手动确认 |
| `angled` | 斜角缺口 | 手动确认 |
| `small` | 小缺口 (<2mm) | 自动吸附 |

**分层补全策略**:
1. **端点吸附 (Snap)**: 端点距离<ε时合并
2. **短缺口桥接 (Auto Bridge)**: 缺口短且方向一致时自动补线
3. **门洞语义补全**: 用户指定缺口为"封闭边界"或"开口边界"
4. **受控绘制**: 提供吸附、正交约束工具手动补全

### 2.5 几何验证服务 (Validator Service)

**功能描述**: 验证几何质量，确保可仿真

**验证项**:

| 验证项 | 错误代码 | 说明 |
|--------|----------|------|
| 闭合性检查 | E001 | 环首尾误差检测 |
| 自相交检测 | E002 | 避免蝴蝶结多边形 |
| 孔洞包含关系 | E003 | holes 必须位于 outer 内部 |
| 微小特征检测 | W001/W002 | 短边、尖角清理 |
| 单位标定 | W003 | 若无单位，要求两点标定 |

**错误恢复建议系统** (P11 特别肯定):

系统为每个错误提供可操作的修复建议：

```json
{
  "code": "E001",
  "message": "环未闭合，端点间距 0.8mm",
  "suggestion": {
    "action": "建议：1) 调整端点位置 2) 增大闭合容差 3) 使用 InteractSvc 桥接缺口",
    "config_change": {
      "key": "validator.closure_tolerance_mm",
      "value": 0.45
    },
    "priority": 10
  }
}
```

### 2.7 声学分析服务 (Acoustic Service)

**功能描述**: 提供声学特性分析和计算

**分析类型**:

#### 1. 选区材料统计 (SELECTION_MATERIAL_STATS)

计算选定区域内的材料分布和等效吸声面积：

```json
// 请求
{
  "type": "SELECTION_MATERIAL_STATS",
  "boundary": {
    "type": "RECT",
    "min": [0.0, 0.0],
    "max": [10.0, 10.0]
  },
  "mode": "SMART"
}
```

**响应内容**:
- 表面 ID 列表
- 总面积
- 材料分布（面积、百分比）
- 等效吸声面积（125Hz-4kHz）
- 平均吸声系数

#### 2. 房间混响时间计算 (ROOM_REVERBERATION)

计算房间的混响时间 T60 和早期衰变时间 EDT：

```json
// 请求
{
  "type": "ROOM_REVERBERATION",
  "room_id": 0,
  "formula": "SABINE",
  "room_height": 3.0
}
```

**支持公式**:
| 公式 | 适用条件 | 计算公式 |
|------|----------|----------|
| `SABINE` | α < 0.2（低吸声房间） | T60 = 0.161 × V / A |
| `EYRING` | α > 0.2（高吸声房间） | T60 = 0.161 × V / (-S × ln(1-α)) |
| `AUTO` | 自动选择 | 根据平均吸声系数自动选择 |

**输出频率**: 125Hz, 250Hz, 500Hz, 1kHz, 2kHz, 4kHz（倍频程）

#### 3. 多区域对比分析 (COMPARATIVE_ANALYSIS)

对比不同区域的声学特性：

```json
// 请求
{
  "type": "COMPARATIVE_ANALYSIS",
  "selections": [
    {
      "name": "区域 A",
      "boundary": {"type": "RECT", "min": [0,0], "max": [10,10]}
    },
    {
      "name": "区域 B",
      "boundary": {"type": "POLYGON", "points": [[0,0],[20,0],[20,15],[0,15]]}
    }
  ],
  "metrics": ["AREA", "AVERAGE_ABSORPTION", "EQUIVALENT_ABSORPTION_AREA"]
}
```

## 三、配置管理

### 3.1 预设配置

系统提供 5 个场景化配置预设，无需手动编写配置文件：

| 预设名称 | 适用场景 | 特点 |
|----------|----------|------|
| `architectural` | 建筑图纸 | 高精度端点吸附，严格闭合检查 |
| `mechanical` | 机械图纸 | 小线段过滤，尖角检测 |
| `scanned` | 扫描图纸 | 宽松端点吸附，自动矢量化 |
| `quick` | 快速原型 | 最小处理，快速输出 |
| `custom` | 自定义 | 用户自定义参数 |

### 3.2 配置参数

**拓扑构建参数**:
```toml
[topology]
snap_tolerance_mm = 0.5          # 端点吸附容差
min_line_length_mm = 1.0         # 最小线段长度
merge_angle_tolerance_deg = 5.0  # 合并角度容差
max_gap_bridge_length_mm = 2.0   # 最大缺口桥接长度
```

**验证参数**:
```toml
[validator]
closure_tolerance_mm = 0.3       # 闭合性容差
min_area_m2 = 0.5                # 最小面积
min_edge_length_mm = 100.0       # 最小边长
min_angle_deg = 15.0             # 最小角度
```

**导出参数**:
```toml
[export]
format = "json"                  # 导出格式
json_indent = 2                  # JSON 缩进
auto_validate = true             # 自动验证
```
