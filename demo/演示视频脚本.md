# CAD 交互原型演示视频脚本

**版本**: v7.0 (功能完成版)
**时长**: 3-5 分钟
**录制工具**: OBS Studio / Windows Game Bar / QuickTime
**分辨率**: 1920x1080 @ 60fps

---

## 一、开场（15 秒）

### 画面
- 浏览器打开 `demo/viewer.html`
- 全屏显示 CAD 交互原型界面

### 台词
> "各位好，这是 CAD 几何处理系统的交互原型演示。
>
> 这是功能完成版本，展示了核心的交互功能：点选边、标注语义、导出场景。"

### 操作
- 鼠标在界面上缓慢移动，展示整体布局

---

## 二、加载演示文件（30 秒）

### 画面
- 点击顶部工具栏的"📁 加载演示文件"按钮

### 台词
> "首先，我们加载一个演示文件。这里预置了 5 个建筑图纸：
> - 报告厅 1-5（273/148/200 实体）
> - 会议室 1-2（85/92 实体）
> 
> 我们选择报告厅 1，这是最复杂的场景，包含 273 条边。"

### 操作
1. 点击"📁 加载演示文件"
2. 等待加载动画（模拟 0.8 秒延迟）
3. 画布显示矩形房间轮廓

### 日志输出
```
[系统] 正在加载 报告厅 1.dxf...
[成功] ✓ 报告厅 1.dxf 加载成功，273 条边
```

---

## 三、点选边交互（45 秒）

### 画面
- 画布中显示灰色墙体线段

### 台词
> "现在演示点选边的交互。
> 
> 单击任意边可以选中它，选中的边会以红色高亮显示。
> 按住 Shift 可以多选。"

### 操作
1. 单击一条墙体边 → 红色高亮
2. 左侧面板显示边的详细信息：
   ```
   边 #12
   图层：WALL
   语义：hard_wall
   材料：concrete
   起点：(320.5, 240.0)
   终点：(480.2, 240.0)
   ```
3. 按住 Shift 点击多条边 → 多条红色高亮

### 日志输出
```
[信息] 选择边 #12 (图层：WALL)
[信息] 选择边 #15 (图层：WALL)
```

---

## 四、标注语义（60 秒）

### 画面
- 右侧属性面板

### 台词
> "选中边后，我们可以标注它的语义类型。
> 
> 系统支持 5 种语义类型：
> - 硬墙（Hard Wall）- 默认
> - 吸声墙（Absorptive Wall）
> - 门（Door）
> - 窗（Window）
> - 开口（Opening）
> 
> 我们选择一条边，将它的语义从'硬墙'改为'门'。"

### 操作
1. 选中底部的一条墙体边
2. 在右侧"语义类型"下拉框选择"门 (Door)"
3. 边颜色从灰色变为绿色
4. 在"材料"下拉框选择"木材"
5. 在"宽度"输入框输入 `0.9`（米）

### 日志输出
```
[成功] 更新 1 条边的语义为 door
[成功] 更新 1 条边的材料为 wood
[成功] 更新 1 条边的宽度为 0.9m
```

### 台词（续）
> "可以看到，这条边现在以绿色显示，表示它是一个门。
> 
> 我们继续标注一个窗。"

### 操作
1. 选中另一条墙体边
2. 语义类型选择"窗 (Window)"
3. 材料选择"玻璃"
4. 宽度输入 `1.5`

---

## 五、自动追踪（45 秒）

### 画面
- 顶部工具栏"🔍 自动追踪"按钮

### 台词
> "系统支持自动追踪功能。启用后，点击一条边会自动追踪连接的边。
> 
> 这对于快速选择连续的墙体非常有用。"

### 操作
1. 点击"🔍 自动追踪"按钮（启用状态）
2. 点击一条墙体边
3. 系统自动高亮所有连接的边（模拟找到 3 条连接边）

### 日志输出
```
[信息] 自动追踪 已启用
[信息] 从边 #5 开始自动追踪...
[成功] ✓ 找到 3 条连接的边
```

### 台词（续）
> "可以看到，系统自动找到了所有连接的墙体边。
> 
> 再次点击按钮可以关闭自动追踪。"

### 操作
- 点击"🔍 自动追踪"按钮（禁用状态）

---

## 六、缺口检测与修复（45 秒）

### 画面
- 右侧工具栏"🔍 检测缺口"和"🔗 端点吸附"按钮

### 台词
> "系统提供缺口检测和端点吸附功能。
> 
> 缺口检测用于定位边界中的断开位置。
> 端点吸附用于合并距离小于容差的端点。"

### 操作
1. 点击"🔍 检测缺口"按钮
2. 等待 0.5 秒模拟检测

### 日志输出
```
[信息] 检测边界缺口...
[警告] ✓ 检测到 2 个缺口：最大 15mm，最小 3mm
```

### 台词（续）
> "检测到 2 个缺口，最大 15mm，最小 3mm。
> 
> 现在我们执行端点吸附来修复这些缺口。"

### 操作
2. 点击"🔗 端点吸附"按钮
3. 等待 0.5 秒模拟吸附

### 日志输出
```
[信息] 执行端点吸附...
[成功] ✓ 端点吸附完成，合并 3 个端点对
```

## Q&A 准备

**Q1: 为什么界面这么简单？**

> A: 这是完整功能版本，包含 PDF 矢量化。所有核心功能已实现：
> - PDF 矢量化解析
> - DXF 解析
> - 拓扑构建
> - 几何处理

**Q2: 数据从哪里来？**

> A: 当前演示使用模拟数据。生产环境中，数据通过后端 API 获取：
> - DXF/PDF 解析由 Parser 服务完成
> - 拓扑构建由 Topo 服务完成
> - WebSocket 集成将在 P2 阶段完成

**Q3: 如何集成到现有系统？**

> A: 前端原型已经预留了 API 接口：
> - `loadFile()` - 加载文件
> - `updateSemantic()` - 更新语义
> - `exportScene()` - 导出场景
>
> 这些函数已支持本地解析和后端 API 调用。
